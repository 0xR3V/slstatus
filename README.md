# Installation

## Dependencies
- `ttf-font-awesome` and `ttf-nerd-fonts-symbols`.

After installing the dependencies, install my build of slstatus:

```sh
git clone https://github.com/0xR3V/slstatus.git
cd status
sudo make clean install
```
